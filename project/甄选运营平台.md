## **1. 项目初始化**

### 1.1 环境准备

- node v16.14.2
- pnpm 8.0.0

### 1.2 初始化项目

本项目使用vite构建，使用 pnmp：performant npm（高性能的 npm）

pnpm安装：

```cmd
npm i -g pnpm@8.0.0
```

项目初始化命令：

```cmd
pnpm create vite
```

选择Vue+TS模板创建项目，进入到项目目录：

```cmd
安装全部依赖
pnmp install 

运行程序
pnpm run dev
```

运行完毕可在 http://localhost:5173/ 上访问项目
