### SAKANA 甄选运营平台

Vue3+TypeScript+SpringBoot 前后端分离项目，功能涵盖数据大屏、权限、商品SKU和SPU等业务

此为前端代码，涉及`vue3+TypeScript+vue-router+pinia+element-plus+axios+echarts`等技术栈
